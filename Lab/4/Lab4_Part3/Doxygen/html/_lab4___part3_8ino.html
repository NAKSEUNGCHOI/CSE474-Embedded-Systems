Detailed Description

running a speaker, an external LED, and FFTs using free-RTOS.

Author
Nakseung Choi 1572578 Kejin Li 1978130
Date
28-May-2022

Detailed Description: In this demo, a real-time pre-emptive operating system is employed to run three tasks:

  1. flash an external LED, one for 100ms and off for 200ms.
  2. speaker that plays "Close Encounters Theme"
  3. measure the performance of an existing FFT function based on a random signal the purpose of this demo is to verify students understanding of free RT-OS. All of these tasks are running concurrently in real time operating system.
